# BELLATRIX: A Reinforcement Learning Hackathon

Welcome to BELLATRIX, the premier hackathon for aspiring AI developers at Cognizance, IIT Roorkee.

## Introduction
Innovative thinking and algorithmic precision are put to the test at BELLATRIX. Participants must optimize intelligent agents for a mystery system in real-time.

## Event Overview
BELLATRIX is a Reinforcement Learning (RL) competition where teams design and tune control systems. The environment codebase is revealed at the start of the event.

### Key Objectives
- Apply RL concepts to control problems.
- Design efficient reward functions.
- Build agents that generalize across stochastic environments.
- Maintain clean, modular, and reproducible code.

## How to Proceed as a Host

### 1. Project Structure
The project is organized into:
- `environment.py`: Defines the challenge (e.g., a control system).
- `agent_template.py`: The base class for participant agents.
- `evaluate.py`: The script to run evaluations and score submissions.
- `leaderboard/`: Directory to store submission results and the current rankings.

### 2. Setting Up the Environment
Ensure Python 3.8+ is installed. Install the required libraries:
```bash
pip install numpy gymnasium matplotlib
```

### 3. Running the Event
1. **Release the Codebase**: Share the `environment.py` and `agent_template.py` with participants at the start of the 12-hour window.
2. **Submissions**: Participants submit their `agent.py` and reward logic.
3. **Automated Evaluation**: Run the `evaluate.py` script against the submissions to update the leaderboard.
4. **Leaderboard**: View rankings using `leaderboard.py`.
